    Load extension locally

    The static demo proves the runtime contract without extension privileges. If you want PNG copy/save flows or capture-backed audit bundles, load the extension locally and enable file URL access when needed.

    1. Start a local preview
    Use a local HTTP server for the most reliable same-origin demo path.
    python3 -m http.server 8765
    2. Load the unpacked extension
    Open chrome://extensions, enable Developer mode, then choose this repo root with Load unpacked.
    3. Allow local files if needed
    When opening file:///… pages directly, turn on Allow access to file URLs for the extension.
    4. Return here for proof
    Once extension runtime is available, bundle export becomes active in this demo.
