The idea

Every page has a structure. Most of the time you can't see it.

a11y-overlay is a small script that draws the invisible scaffolding of a webpage on top of itself — live, over real CSS, with no DOM mutations on the host page. It was built to make one specific thing easier: looking at a page and immediately knowing what's a landmark, what's clickable, what's a form, and in what order a keyboard will traverse it. Turns out that's useful for humans doing accessibility work, and for agents deciding what to click.

01 — Zero instrumentation

Runs on any webpage, as-is.

No attributes, no build step, no config. Paste a script tag or install the extension and it reads the live DOM through getBoundingClientRect + standard selectors.

02 — Nothing gets mutated

The host page is untouched.

All chrome lives inside a shadow root on a single fixed-position host <div>. No styles injected into the page, no attributes added to its elements, no layout side-effects.

03 — One visual language

Colors mean the same thing, everywhere.

Amber is always a landmark. Cyan is always a heading. Lime is always interactive. Once you learn the palette you can read any annotated page in seconds.

Eleven toggles

Everything is a single keypress.

The overlay has no mouse UI. Once injected, press ? for help, or any of the eleven single-letter toggles to turn a layer on and off. Press X to kill the overlay entirely.

Each layer adds one kind of information. Stack them together for a dense audit; peel them back one at a time to read the page.

Inspector mode

Every label opens a receipt.

Click any badge or overlay label and a Selection Inspector opens in the bottom-left. It shows what was flagged, why, how confident the finding is, and — crucially — which rule it came from. Nothing is pulled out of a hat.

Finding types
Standards-based
Direct rule from WCAG, ARIA-APG, or the HTML spec. Deterministic pass/fail.
Heuristic
Pattern-matched observation (repeating siblings, small tap targets). Flagged, not failed.
Mixed
A rule plus a heuristic. E.g. form-label-weak — a control has a name, but only via placeholder.

Use modes

Two audiences. One proof surface.

Open any page. Press a letter.

No dev-tools panel, no Figma plugin, no dashboard to set up. The overlay lives where you already are — in the page itself.

  • Audit any site in 30 seconds. Flip L H F on, scroll, done.
  • Spot small tap targets instantly. T rings every hit area under 24×24 px.
  • Flag missing alt text. A highlights every decorative-vs-content image mismatch.
  • Drop notes right on the page. N for sticky, W for an arrow. Export as PNG for the ticket.
  • No network, no telemetry. Runs entirely in the tab's isolated world.

01 — BROWSER

Chrome extension

Load unpacked from the repo root. Click the toolbar icon, or hit ⌘⇧Y on mac, Ctrl⇧Y elsewhere.

# chrome://extensions → Developer mode
# → Load unpacked → select repo folder
→ allow access to file URLs
#   (for local file:// pages)
MV3 · ACTIVETAB · LOCAL-ONLY
02 — ANY PAGE

Script tag or bookmarklet

Drop the file into a page you own, or drag it to your bookmarks bar for one-click injection elsewhere.

<script src="a11y-overlay.js"></script>
// or: bookmarklet, same file, no deps
ONE FILE · NO DEPS
Heads up — strict-CSP sites reject both bookmarklets and script tags. Use the extension there.
03 — AGENT

Playwright / Puppeteer

Inject at context-load so every page the agent navigates to comes pre-annotated, then read the public API at window.__a11yOverlayInstalled.

await page.addInitScript({
  path: './a11y-overlay.js'
});
await page.waitForFunction(
  () => window.__a11yOverlayInstalled
);
PROOF SURFACE · DETERMINISTIC
04 — SKILL

Agent skill

Install overlay-playwright-runtime for local audits, authenticated browser sessions, visual evidence, report bundles, and reviewable annotation previews.

# use --agent codex for Codex installs
npx skills add https://github.com/ranvier2d2/a11y-overlay \
  --full-depth \
  --skill overlay-playwright-runtime \
  --agent codex
Add --global --yes only when you explicitly want a non-interactive global install.
