a11y-overlay

Privacy Policy

Last updated: April 18, 2026

a11y-overlay does not collect, transmit, sell, or share personal data.

What the extension does

a11y-overlay injects a local JavaScript file into the page you are currently viewing so it can draw accessibility and structure overlays directly in your browser.

Data collection

The extension does not send page content, browsing history, account information, form data, or personal data to any server. It does not use analytics, advertising, or remote tracking.

Data storage

The extension does not store user data remotely. It also does not require account creation, sign-in, or background sync.

Permissions

The extension requests activeTab and scripting so it can inject a11y-overlay.js into the current tab when the user explicitly clicks the extension action or uses the assigned shortcut. It also requests clipboardWrite so a focused extension export window can copy locally generated PNG exports to the clipboard, downloads so it can save locally generated PNG exports to disk when the user asks for that, and storage so the focused export window can load the most recent export context without sending data to a server.

Third-party services

The extension does not depend on third-party APIs, SaaS services, remote code, or external telemetry.
